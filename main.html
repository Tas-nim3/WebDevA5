Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : It helps to set the html inside an element . innerText : It helps to set the text inside an element . textContent : sets all text inside an element.

Question-3: What is event delegation in the DOM?

Event delegation in Dom is a technique where a parent handle events for its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling means when an even occurs on an element , if first runs on that element and then moves up to its ancestor.
